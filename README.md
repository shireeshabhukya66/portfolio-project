# Full Stack Portfolio Website

A modern full stack portfolio website built using React.js, Tailwind CSS, Django REST Framework, and MySQL.

## Features
- Responsive modern UI
- React.js frontend
- Django REST API backend
- MySQL database integration
- Contact form with API integration
- Axios API requests
- Tailwind CSS styling
- GitHub & LinkedIn integration
- Responsive design for all devices

## Tech Stack
Frontend:
- React.js
- Tailwind CSS
- Axios

Backend:
- Django
- Django REST Framework

Database:
- MySQL

## Functionalities
- Portfolio sections
- Skills showcase
- Projects showcase
- Contact form
- Backend API integration
- Database storage for messages
